# JPostman.io React Site

React + Vite landing page for `jpostman.io`.

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages deployment

This project is ready for a custom domain:

```text
public/CNAME -> jpostman.io
```

Recommended GitHub Pages setup:

1. Create a GitHub repository, for example `JPostman/jpostman.io` or `JPostman/jpostman-site`.
2. Push this project.
3. Run `npm install`.
4. Use GitHub Actions or `npm run deploy` to publish the `dist` folder.
5. In repository settings, enable GitHub Pages.
6. Add `jpostman.io` as the custom domain.
7. In your DNS provider, point `jpostman.io` to GitHub Pages.
8. Enable HTTPS after DNS verification completes.
